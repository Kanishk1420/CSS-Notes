Chapter 6 - CSS Flexbox

  • Float property:- The float property is simple. it just gives flows the element towards left/right.mostly used to Design Profile

  • Clear property:- Used to clear the float. it specifies what elements can float beside a given element.

    • Flexbox provides us with a very simple way to create layouts,align and distributive space among items in a container.

  • Flex-Direction Property:- Defines the direction towards which items are laid can be Row,Column and reverse.

Flex Properties for Parent or Flexbox

  • Following are the properties for the Flex Parent:-

    • Flex-wrap: Can be wrap,no-wrap,wrap-reverse,Wrap-items as needed with this property.

    • *Justify-Content:-Defines aligment along main axis.

    • Align-items: Defines aligment along cross axis.

    • Align-Content: This added no-space if a Flexbox is in wrap align-items but align items in center added gap in between when you wraped it. so align-content fixed that it created no-space btw flex-items.

    Flex Properties for Children or Flex-items

  • Following are the properties for the Flex Children:-

    • Order: Controls the order in which the items appear in the flex container. ex:- 1,2,3

    • Align-self: Allows default aligment to be overriden for the indiviual flex items. ex:- like flexbox already in center but in align-items the items contain in it use any property it want like flex-end and start etc.

    • Flex-grow: Defines the ablity for a flex items to grow like 1. ex:- You can increase the width and height of flex-item inside Flexbox.
