Topic 8 :- Transforms Transitions & Animations

  • The transform CSS property lets you rotate, scale, skew, or translate an element. It modifies the coordinate space of the CSS visual formatting model. It used to apply 2D and 3D transformation to an element.

  • 2D transform:- Used to apply to change x and y axis.

  • 3D transform:- Used to apply to change x,y and z axis.

Types of 2D & 3D Transform Properties

You can use the following 2-D & 3-D Transforms in CSS:

  1. Translate:- Used to shift the element of x-axis and y-axis.

  2. Rotate X,Y,Z:- Used to rotate the element.

  3. Scale X,Y,Z:- Used to scale the element into X and Y axis.

  4. Skew:- Used to scale the element as in both axis in z axis mainly used for 3d objects.

  • Transform-Origin is used where you want to apply the transform onto it.

CSS Transitions

  • Used to change property values smoothly, over a given duration.

    • The Transition Property

  • The Transition property is used to add transitions in CSS.

    • Following are the properties used for CSS transitions.

  1. Transition-property:- the property you want to add transitions. like width

  2. Transition-duration:- the property you want to apply transitions. like 3s

  3. Transition-timing-function:- How you want the property to transition. like ease in,out

  4. Transition-delay:- Specifes the delay for transition.

    5. In Case use transition itself to use all Properties in one.

CSS Animations

  • Used to animate CSS Properties with more Control we can use @keyframes rules to chnage the Animation from a given style to a new style.

  • keyframes name{ from{ width:20px } to{ width:30px } }

    • The Animation Property

  • The Animation property is used to add Animations in CSS.

    • Following are the properties used for CSS Animations.

  1. Animation-name:- the property used to name the Animation.

  2. Animation-duration:- How long does the Animation run?

  3. Animation-timing-function:- Determines speed curve of Animation. like ease in,out,etc

  4. Animation-delay:- Specifes the delay for Animation.

  5. Animation-iteration-count:- Number of times an Animation should run.

  6. Animation-direction:- Specifes the direction of the Animation.

    7. In Case use Animation itself to use all Properties in one.

    • You can also use % values to indicate what should happen when a certain percent of animation is completed. Like 0%{}, 50%{}, 100%{}
