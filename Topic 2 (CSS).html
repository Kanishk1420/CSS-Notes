  • Types of Color values

  1. RGB:- specify color using red,green,blue values eg:- rgb(280,98,70),where 280 represents red, 98 represents green & 70 represents blue.

  2. HFX Code :- Specify Color suing hex code eg:- #ff7180

  3. HSL Code:-Specify the color using HSL values eg:- hsl(8,90%,63%)

  • the value of the color or background color is provided as any one of these values.

  • We also have an RGBA and HSLA values for color but they are rarely used by beginners, A stands for Alpha.

Topic 2.1- Properties of CSS

  • The CSS backgound colour property specifies the background color of a container.

  • ex:-.red { background-color: red;}

  • The CSS background image property specifies the image as the background .

  • ex:-body{background-image:url("alhaitham.jpg")}

    • The Background Repeat Property

  • Can be any of:

  • Repeat-x :- Repeat in Horizontal Direction

  • Repeat-y :- Repeat in Vertical Direction

  • No-Repeat :-Image not repeat

    • The Background-Size property

  • Can be following:

  • Cover:- fits & no empty space requires.

  • Contain:-fits & image is fully visible.

  • Auto:-Display in original Size.

  • {{width}}:-Set width & height will be set automatically.

    • The Background-Position property

  • Sets the starting positon of a background image.

  • Ex:- body{background-position: left top;}

    • The Background-Attachment property

  • Defines a scrollable/non-scrollable Character of a backgound image.

  • background-attachment:fixed

    • The Background Shorthand

  • A single property to set multiple background Properties

  • body{background:red url() no-repeat fixed right top;}

  • Here 'Red' represents Color 'url()' represents image 'no-repeat' represents repeat of background 'fixed' represents attachment'right top' represents positon.
